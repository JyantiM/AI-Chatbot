# AI Chatbot Application

## Project Overview

The AI Chatbot Application is a full-stack web platform that enables users to interact with an intelligent conversational assistant, generate AI-based images, purchase usage credits, and share generated images within a community gallery.

The application is designed with a responsive user interface supporting both mobile and laptop views, includes dark mode, and ensures secure authentication and payment handling.

## Core Capabilities

- AI-powered chat with formatted responses
- Code-formatted answers using Markdown and syntax highlighting
- Secure user authentication (Login and Registration)
- Credit-based usage system
- AI image generation and publishing
- Community image gallery
- Responsive design for mobile and desktop
- Dark mode support
